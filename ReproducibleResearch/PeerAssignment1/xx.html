Loading and preprocessing the data

Read the CSV file. Convert the data frame to a data table using the [data.table] Convert the “date” variable to a “date class”.

library(data.table)
dt <- read.csv(file.path(getwd(), "activity.csv"))
dt <- data.table(dt)
dt <- dt[, date := as.Date(date)]
